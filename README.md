# BoType: 用于 Microsoft Word 的高效率公式助手

BoType 是一款专门为撰写带编号数学公式的科研工作者、学生和工程师开发的 Microsoft Word 插件。由于它使用 VSTO（.NET）进行深层开发，因此比传统的 VBA（.ppam, .dotm）具有更高的运行效率和稳定性。

**注意**：本插件并非如 IguanaTex 那般将 LaTeX 渲染为图片插入，而是通过原生的 OMath 对象对 Word 内部机制进行优化，排版和交叉引用均为 Word 标准格式！

## 主要功能

*   **极速单行带编号公式 / 行内公式**：一键在正文插入预先划定好三栏不可见表格（用于绝对居中）或原生行内公式框。
*   **按要求自定义多级/单数编号样式**：支持类似 `(1)`、`Equ. 1-1` 或是 `公式 1.1` 的灵活章节序号支持。
*   **快速对已选中公式实施包裹/编号**：将没有编号的草稿公式瞬间包裹并编号。
*   **LaTeX 原生转录支持**：通过 Office 新版本的接口支持 LaTeX 书写快速一键转化为结构化数学方程，支持继续编辑！
*   **自动化引用于同步刷新**：自动维护 `OLE_LINK` 书签并能按指定引用样式通过 `STYLEREF` 与 `SEQ` 域随时完成精准刷新，彻底告别“前文修改后文引用全打乱”的通病。

## 软件安装 (以 2.x Release 版本为例)

由于使用了 VSTO 技术开发，该程序编译产物为 .dll（比传统 VBA 宏文件更底层和强大）。

### 1. 下载 Release 压缩包

前往 [Releases 页面](https://github.com/bo-qian/BoType/releases) 或者获取最新的打包文件 `BoType_Release.zip`。

### 2. 解压并运行

1.  将下载好的 `.zip` 文件 **完全解压** 到本地文件夹（请勿在压缩包内直接双击！）。
2.  进入解压后的文件夹内，双击运行 **`BoType.vsto`** 或者 **`setup.exe`**。
3.  在弹出的 Office 安装向导中，一直点击 **“安装”/“下一步”** 即可。

*注意：VSTO 插件采用的是微软的 ClickOnce 部署技术，它会自动静默安装到 Windows 的应用缓存目录 (`%LocalAppData%\Apps\2.0\`)，这是无法在安装时手动选择硬盘位置的，是微软底层的安全设计机制。你在哪里双击安装都没有关系，装完就可以删掉安装包文件。*

### 3. 打开 Word

重新启动打开你的 Microsoft Word。你会看到顶部的功能区多出了一个 **"BoType"** 标签卡。享受行云流水的公式输入吧！

## 常见问题与排错 (FAQ)

### 为什么安装时提示 "未授予信任" 或 "Publisher 无法验证"，并且安装按钮是灰色的报错被拦截？
这是由于安装包目前未签署付费的商业数字签名证书导致被微软 Office 拦截。如果你遇到了红色的禁止安装报错：
1. 请在解压的文件夹中找到 `BoType_TemporaryKey.cer` 文件。
2. 双击打开它，点击“安装证书”。
3. 存储位置选择“**本地计算机**”（需要管理员权限）。
4. 将证书放置于下列存储：**受信任的根证书颁发机构**（再重复一遍放在 **受信任的发布者**）。
5. 完成后，重新双击 `BoType.vsto` 即可正常安装。

### 为什么在 Word 中看不到 BoType 标签卡？
有可能是在使用过程中意外崩溃导致插件被“禁用加载项”拦截。你可以前往：
Word 内点击 **文件** -> **选项** -> **加载项** -> 最下方管理：**COM 加载项** -> 点击“**转到...**” -> 勾选 `BoType` 复选框 -> 确定。
